Detailed Description

DYN-DI/DYN-CG Program responsible to calculate A link speed and displacement by Newmark method, and add up to get final speed/displacement.

This program's input file description can be found at the documentation of the function carl::get_input_params(GetPot& field_parser, feti_loop_dyn_params& input_params).

In this step, it will use the following files in scratch foler:

  • this_acc_A_link_sys_sol_vec.petscvec : \( \ddot{U}^A_{link}(t) \)
  • this_acc_A_free_sys_sol_vec.petscvec : \( \ddot{U}^A_{free}(t) \)

It will create:

  • this_speed_A_link.petscvec: \( \dot{U}^A_{link}(t) \)
  • this_disp_A_link.petscvec: \( U^A_{link}(t) \)
  • this_acc_A.petscvec : \( \ddot{U}^A(t) \)
  • this_speed_A.petscvec: \( \dot{U}^A(t) \)
  • this_disp_A.petscvec: \( U^A(t) \)

At the end, it will apply a test, move all this to prev for object A, rewrite iteration_progression.txt and set all A/interpolation related vector to NAN.

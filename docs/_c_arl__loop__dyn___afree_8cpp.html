Detailed Description

DYN-DI/DYN-CG Program responsible to calculate A free speed and displacement by Newmark method.

This program's input file description can be found at the documentation of the function carl::get_input_params(GetPot& field_parser, feti_loop_dyn_params& input_params).

In this step, it will use the following files in scratch foler:

  • prev_acc_A.petscvec : \( \ddot{U}^A(t-\Delta T) \)
  • this_acc_A_free_sys_sol_vec.petscvec : \( \ddot{U}^A_{free}(t) \)
  • prev_speed_A.petscvec : \( \dot{U}^A(t-\Delta T) \)
  • prev_disp_A.petscvec : \( U^A(t-\Delta T) \)

It will create:

  • this_speed_A_free.petscvec: \( \dot{U}^A_{free}(t) \)
  • this_disp_A_free.petscvec: \( U^A_{free}(t) \)
